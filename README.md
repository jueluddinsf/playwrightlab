# PlaywrightLab Landing Page

A high-performance, single-page landing page for "PlaywrightLab LLC," a technical consultancy specializing in Playwright automation migrations.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Language:** TypeScript

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/page.tsx`: Main landing page assembly.
- `app/components/`: Reusable UI components.
    - `Navbar.tsx`: Sticky navigation.
    - `Hero.tsx`: Hero section with code comparison.
    - `TechStackTicker.tsx`: Infinite scrolling tech stack marquee.
    - `MigrationProtocol.tsx`: Problem/Solution section.
    - `Services.tsx`: Bento Grid of services.
    - `Footer.tsx`: Site footer.
- `app/globals.css`: Global styles and Tailwind configuration.

- **Interactive Elements:** Smooth scrolling, hover effects, and animations.
